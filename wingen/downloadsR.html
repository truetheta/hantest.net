Note:
If your system has Windows XP Family and has never installed Microsoft .NET framework on your system before, it is necessary to install .NET framework first.
If you are not sure whether your system has .NET framework or not, just run ’setup.exe’ file in 'WinGen_Han.zip'. The installation program will automatically check your system and download .NET framework from Microsoft website if your system does not have it. However, it could take several minutes (up to 30 minutes).
