About IRTEQ

Maintaining comparability of scores from different forms of a test (and/or from different test administrations) has been a major technical challenge for psychometricians. Today, with a major focus of state testing programs on the assessment of growth, the challenge of maintaining comparability of scores has become even more important. Over the years, several IRT-based scaling/equating methods have been developed to provide solutions (e.g., Hambleton, Swaminathan, & Rogers, 1991; Kolen & Brennan, 2004; Lord, 1980).

While equating methods research has flourished, software development has been limited. The few computer programs for test scaling and equating that have been developed for wide use do not always include features of special interest to researchers — for example, available software cannot handle all the popular IRT models being applied to test data, and cannot handle some of the popular equating designs. IRTEQ was developed to address that need.

1. Various IRT Scaling Methods

IRTEQ can rescale a test form to another using widely used IRT scaling/equating methods with an anchor test design: Mean/Mean (Loyd & Hoover, 1980), Mean/Sigma (Marco, 1977), Robust Mean/Sigma (Linn, Levine, Hastings, & Wardrop, 1981), and TCC methods (Haebara, 1980; Stocking & Lord, 1983). For TCC methods, IRTEQ provides the option to choose various score distributions for incorporation into the loss function.

2. Support for Various IRT Models

IRTEQ supports various popular unidimensional IRT models: logistic models for dichotomous responses (with 1, 2, or 3 parameters), the Generalized Partial Credit Model (GPCM), Partial Credit Model (PCM), and Graded Response Model (GRM) for polytomous responses. A mixture of IRT models is also supported, and the number of response categories can vary. There is practically no limit to the number of items in each form or the linking item set (>10,000,000). IRTEQ can import item parameters and score data directly from WinGen and/or PARSCALE.

3. True Score Equating

In addition to test item scaling, IRTEQ implements true score equating. Raw scores or test scores from one test form can be linked to another test form, and a test score to test score conversion table is provided using true score equating (see Lord, 1980).

4. User-Friendly Interface & Research Tools

As a Windows application, IRTEQ employs an intuitive, user-friendly graphical user interface (GUI), so the whole process of scaling and equating can be done with a few point-and-clicks. IRTEQ can also be operated with syntax files and cue files for more intensive uses. Plots comparing a, b, and c-parameter estimates of the linking items across test forms are provided. Test characteristic curves for each test form, the linking item set, and the rescaled test form are also provided.
