About AllTestSim

AllTestSim is a next-generation test simulation platform that brings WinGen + MSTGen + SimulCAT + a lot more, all into one unified environment. Building on over a decade of development experience, AllTestSim eliminates the need to switch between separate tools — enabling IRT data generation, CAT simulation, MST simulation, and beyond within a single, integrated framework.

Whether you are conducting fundamental research, evaluating operational test programs, or teaching adaptive testing concepts, AllTestSim provides a comprehensive, modern platform for all your test simulation needs.

Planned Features

Unified CAT & MST Engine

Run CAT and MST simulations side by side using a shared simulation core, with consistent configuration and output formats.

Modern Architecture

Built on .NET 10 with both GUI and command-line interfaces. Cross-platform console engine for Windows and macOS.

Advanced Exposure Control

Comprehensive exposure control methods including Local Fade-Away (LFA) and panel-based designs for both CAT and MST modes.

Comprehensive IRT Support

Full support for dichotomous (1PL, 2PL, 3PL) and polytomous (GRM, GPCM, PCM, NRM) IRT models with mixed-model capability.

Batch & Scripting

Syntax file support and command-line interface for large-scale simulation studies and automated workflows.

Visual Analytics

Auto-generated plots for exposure rates, score distributions, information functions, and comparison across simulation conditions.
