About

This site distributes psychometric research papers and software that I have authored or co-authored. Everything is free of charge, but the author(s) retain all copyrights. Please cite appropriately.

Software Downloads

For research and teaching purposes, no license is required. For commercial use, please contact me first. The author(s) are not responsible for any results produced by the software and do not provide formal support, but bug reports are appreciated.

  • SimulCAT 2012
    Windows application that simulates computerized adaptive test administration.
    Han, K. T. (2012). SimulCAT: Windows software for simulating computerized adaptive test administration. Applied Psychological Measurement, 36(1), 64–66.
  • IRTEQ 2009
    Windows application that implements IRT scaling and equating.
    Han, K. T. (2009). IRTEQ: Windows application that implements IRT scaling and equating. Applied Psychological Measurement, 33(6), 491–493.
  • WinGen3 2010
    Windows software that generates IRT parameters and item responses.
    Han, K. T. (2010). WinGen3: Windows software that generates IRT parameters and item responses. Applied Psychological Measurement, 34(8), 609–610.
  • MSTGen 2013
    Simulated data generator for multistage testing.
    Han, K. T. (2013). MSTGen: Simulated data generator for multistage testing. Applied Psychological Measurement, 37(8), 666–668.
  • ResidPlots-2 2009
    Computer software for IRT graphic residual analyses.
    Han, K. T., & Hambleton, R. K. (2009). ResidPlots-2: Computer software for IRT graphic residual analyses (Version 2.0) [Computer software].
