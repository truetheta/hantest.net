What's New in Version 2.0

  • Single-engine architecture — GUI and command-line share the same simulation core (SimulCATcon), ensuring consistent results across both modes
  • Command-line interface (SimulCATcon) — Run simulations from the terminal with SimulCATcon.exe syntax.scs; ideal for batch processing and scripting
  • Local Fade-Away (LFA) exposure control — New theta-node-based exposure control method for more uniform item usage across the ability scale
  • Item review and edit simulation — Supports Item Pocket and Moving Window methods for simulating examinee review-and-edit behavior
  • Auto-generated exposure charts — Overall and conditional item exposure rate scatter plots are produced automatically after each simulation
  • Migrated to .NET 10 — Modern runtime with improved performance; cross-platform console engine (Windows and macOS)
  • Automatic update check — SimulCAT notifies you when a new version is available on launch
